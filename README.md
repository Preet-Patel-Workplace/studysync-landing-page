# StudySync – Educational Landing Page

## Overview
StudySync is a responsive educational landing page built using HTML and CSS.  
The project focuses on clean UI design, structured layout, and responsive behavior across different screen sizes.

## Features
- Responsive multi-section layout (Hero, Features, Testimonials, Newsletter, Footer)
- Semantic HTML structure for better accessibility
- Clean and consistent UI styling using modern CSS
- Grid and Flexbox for layout management
- Organized asset management using an images folder

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## What I Learned
- Building real-world landing page layouts
- Managing large UI structures with reusable CSS classes
- Implementing responsive design for mobile and desktop
- Organizing project assets professionally
